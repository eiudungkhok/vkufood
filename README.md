# 🍔 VKU FOOD

Website đặt đồ ăn dành cho sinh viên VKU với giao diện hiện đại, dễ sử dụng và đầy đủ chức năng quản lý tài khoản, đặt món và thanh toán.

## 🚀 Tính năng

### 👤 Tài khoản
- Đăng ký tài khoản
- Đăng nhập
- Đăng xuất
- Khôi phục mật khẩu
- Cập nhật thông tin cá nhân

### 🍕 Đặt món
- Hiển thị danh sách món ăn
- Phân loại món ăn
- Tìm kiếm món ăn
- Chọn số lượng món
- Thêm vào giỏ hàng
- Thanh toán

### 💳 Thanh toán
- Hiển thị hóa đơn
- Tạo mã QR thanh toán
- Sinh mã giao dịch ngẫu nhiên
- Xác nhận địa chỉ giao hàng

### 📜 Lịch sử
- Lưu lịch sử mua hàng
- Xem lại đơn hàng đã đặt

### 🎨 Giao diện
- Responsive trên nhiều thiết bị
- Hiệu ứng động
- Thiết kế hiện đại
- Dễ sử dụng

## ⚙️ Cài đặt

### Clone dự án

```bash
git clone https://github.com/eiudungkhok/VKU-FOOD.git
```

### Chạy dự án

1. Cài đặt XAMPP.
2. Copy thư mục dự án vào:

```text
xampp/htdocs/
```

3. Khởi động Apache.
4. Truy cập:

```text
http://localhost/vkufood
```
